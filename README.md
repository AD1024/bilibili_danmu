# B 站直播弹幕姬 Python 版

Python 抓取 B 站直播弹幕。

---

## 简单说明

B 站弹幕协议是会变的，目前至少改过一次。故不能保证向后兼容性。

---

### 快速开始

在 config.py 中配置是否显示礼物、欢迎信息。
1: 开启
0: 关闭

在命令行中直接执行，
	```
    python3 initalize.py

    ```
该脚本会自动安装所需依赖

根据提示输入房间号即可。
